✅ Justification des Choix Architecturaux

🏗️ Microservices

  • Scalabilité: 150 → 500+ restos
  • Résilience: Isolation pannes
  • Évolutivité: Nouveaux services
  • Équipes: Autonomie

⬡ Hexagonale

  • Testabilité: Cœur isolé
  • Indépendance: Technologie
  • Clarté: Métier/infra
  • Évolution: Adapter métier

⚡ Event-Driven

  • Découplage: Async
  • Performance: Background
  • Traçabilité: Event Store
  • Sync ERP: Temps réel

❌ Pourquoi PAS Monolithique ?

  • Scalabilité insuffisante (2/5) → Impossible 500+ restaurants
  • Maintenabilité faible (2/5) → Reproduirait code legacy
  • Évolutivité limitée (2/5) → Déploiements risqués

🔴 Points Faibles du SI Actuel

Technologie & Code

Critique Technologie obsolete

  • Windows Server 2008 R2 hors support
  • ASP.NET 2005 - framework abandonne
  • Failles de securite non corrigees
Impact: Risque securite majeur

Critique Code non maintenable

  • Patchs sur patchs depuis 15 ans
  • 0% de couverture de tests
  • Couplage fort, pas de modularite
Impact: Cout maintenance x3

Performance & UX

Haute Performance degradee

  • Temps de chargement > 5 secondes
  • Taux d'abandon: 45% vs 30%
  • Pics de charge non geres
Impact: Perte CA -15%

Haute Pas d'app mobile

  • 80% du trafic est mobile
  • Site non responsive
  • Concurrents ont tous une app
Impact: Perte CA -25%

Securite & Conformite

Critique Non-conformite RGPD

  • Donnees personnelles non chiffrees
  • Pas de consentement explicite
  • Pas de droit a l'oubli
Risque: Amende 4% du CA

Critique Pas de PCI-DSS

  • Paiements non securises
  • Taux d'echec: 18% vs 2%
  • Risque de fraude eleve
Risque: Perte confiance

Infrastructure & Integration

Haute Mono-hebergeur (SPOF)

  • Un seul datacenter
  • Pas de redondance
  • Derniere panne: 48h d'arret
Impact: Dispo < 99%

Haute Sync 1x/jour

  • Donnees ERP decalees J-1
  • Erreurs de stock frequentes
  • Commandes sur produits epuises
Impact: Insatisfaction clients

Analyse SWOT

Forces (S)

  • 150 restaurants implantes
  • 80% CA en ligne
  • Marque etablie depuis 1992

Faiblesses (W)

  • SI obsolete (ASP.NET 2005)
  • Pas d'app mobile native
  • Non-conformite RGPD/PCI-DSS

Opportunites (O)

  • Croissance marche livraison
  • Expansion 500+ restaurants
  • Digitalisation franchises

Menaces (T)

  • Concurrents (UberEats...)
  • Amende RGPD (4% CA)
  • Perte clients (UX degradee)
